<tr><td VALIGN=TOP width="626"><h2 align=center>疫情爆发以来美国首次出现银行倒闭</h2>
<img width="600" src="https://i.ntdtv.com/assets/uploads/2020/04/1-40-800x450.jpg" />
<h6>MVB银行收购First State Bank的存款和资产。（shutterstock）</h6>
<hr>
	<p>【新唐人北京时间2020年04月05日讯】4月3日，美国联邦<ahref="https://github.com/tsfcd2943/ntdtv/blob/master/gb/存款.md#1">存款</a>保险公司(FDIC)宣布，接管位于西弗吉尼亚州的<ahref="https://github.com/tsfcd2943/ntdtv/blob/master/gb/银行.md#1">银行</a>The First State Bank of Barboursville（以下简称First State Bank）。西弗吉尼亚州的金融监管部门已关闭该银行，事发突然，此前并无消息向公众传达。这是<ahref="https://github.com/tsfcd2943/ntdtv/blob/master/gb/中共肺炎.md#1">中共肺炎</a>(又称武汉肺炎、新冠肺炎，COVID-19)<ahref="https://github.com/tsfcd2943/ntdtv/blob/master/gb/疫情.md#1">疫情</a>发生以来，美国第一家倒闭的银行，不过，不能确定疫情是压倒骆驼的最后一根稻草。</p>
<p>多位经济学家指出，本轮美国经济波动与2008年最大的不同在于，尚无金融机构破产倒闭，这也是尚不能定义金融危机已来的重要参考。</p>
<p>对于被接管的原因，公告称，由于长期的资本和资产质量问题，自2015年以来First State Bank就一直财务困难。该<ahref="https://github.com/tsfcd2943/ntdtv/blob/master/gb/银行.md#1">银行</a>2019年12月31日的财务报告显示，其资本水平过低，无法继续运营。尚未明确指出新冠<ahref="https://github.com/tsfcd2943/ntdtv/blob/master/gb/疫情.md#1">疫情</a>对该银行经营的冲击。</p>
<p>根据处置方案，First State Bank的储户将自动成为MVB银行的储户，FDIC将继续为<ahref="https://github.com/tsfcd2943/ntdtv/blob/master/gb/存款.md#1">存款</a>提供保险，此外，MVB银行还同意购买第一州银行约1.472亿美元的资产。First State Bank的官网页面目前已显示为MVB银行。</p>
<p>FDIC估计，本次将动用存款保险基金（DIF）的成本将为4,680万美元。他们还称，与其他选择相比，MVB银行的收购是存款保险基金成本最低的解决方案。</p>
<p>First State Bank是西弗吉尼亚州一家地方银行，提供全方位的存贷款服务，有超过100年的历史。</p>
<p>MVB银行是一家总部在西弗吉尼亚州的费尔蒙的区域银行，成立仅23年，由社区银行成长为金融科技见长的大银行，业务遍及中大西洋地区及其他地区。其控股公司MVB Financial Corp，为一家金融控股公司，旗下包括MVB银行及该银行的子公司MVB 抵押，MVB 社区发展公司等。</p>
<p>根据处置方案，First State Bank的储户将自动成为MVB银行的储户，FDIC将继续为存款提供保险，因此客户无需更改银行关系。贷款仍可正常还款，特殊问题可与原贷款员联系。若因银行提供服务或产品而欠款，有资格向First State Bank索赔。</p>
<p>First State Bank的四个分支机构将在美国当地时间4月4日正常营业时间内重新开放为MVB银行的分支机构。First State Bank的客户应继续使用其现有的分支机构，直到他们从MVB银行收到有关其已完成系统更改的通知。</p>
<p>截至2019年12月31日，First State Bank的总资产约为1.524亿美元，存款总额为1.395亿美元。除了承担所有存款外，MVB银行还同意购买第一州银行约1.472亿美元的资产。FDIC将保留剩余资产，以备后续之用。FDIC估计，存款保险基金（DIF）的成本将为4,680万美元。他们还称，与其他选择相比，MVB银行的收购是存款保险基金成本最低的解决方案。</p>
<p>对于接管First State Bank的原因，FDIC的公告称，由于长期的资本和资产质量问题，自2015年以来First State Bank就一直财务困难。该银行2019年12月31日的财务报告显示，其资本水平过低，无法继续运营。</p>
<p>FDIC成立于1933年，1934年1月1日开始提供存款保险。FDIC通过为银行和储蓄机构的存款提供保险、识别和监控存款保险基金中的风险、限制银行和储蓄机构倒闭时对经济和金融体系造成的影响，来维持和提高公众对国家金融体系的信心。</p>
<p>FDIC根据风险资本比例（risk-based capital ratio）将投保银行分成五个等级：资本状况良好：比例为10% 及以上；资本充足：8%及以上；资本不足：小于8%；资本严重不足：小于6%；资本极端不足：小于2%。当银行出现资本极端不足时，FDIC会宣布银行破产并接管银行。</p>
<p><strong>编者按：</strong>中共为推卸责任，正在把武汉新冠病毒的来源嫁祸给其他国家。为了阻止中共的罪行，新唐人认为应该将病毒命名为“中共病毒”。这个病毒来自中共统治下的中国，因中共掩盖疫情而向全世界散发，为了让世人永记中共带给世界的灾难，正名为“中共病毒”。</p>
