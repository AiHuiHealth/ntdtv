<tr><td VALIGN=TOP width="626"><h2 align=center>Win7 不想升级到 Win10？8招安心用旧机</h2>
<img src="https://i.ntdtv.com/assets/uploads/2020/01/GettyImages-92142700--800x450.jpg" />
<h6>Windows 7 不想升级到 Windows 10 怎么办？这 8 招让你安心用旧机！（Oli Scarff / Getty Images）</h6>
<hr>
	<p><a href="https://github.com/jtswrk284/ntdtv/blob/master/gb/微软.md">微软</a>今（14日）正式终止支援 <a href="https://github.com/jtswrk284/ntdtv/blob/master/gb/windows-7.md">Windows 7</a>，宣布 2009 年发布的 Windows 7 正式走入历史，同时微软也建议用户可以透过换机，将电脑替换为 <a href="https://github.com/jtswrk284/ntdtv/blob/master/gb/windows-10.md">Windows 10</a>。不过，不论是已经习惯使用 Windows 7，或是暂时不想更换电脑的用户，可能暂时仍没有动机更新电脑，这时面临微软不再推送安全更新、导致的资安风险，又该如何因应？</p>
<p>国外科技网站便整理了 8 种方式，让用户可以在相对安全的情况下，继续使用自己的旧 <a href="https://github.com/jtswrk284/ntdtv/blob/master/gb/windows-7.md">Windows 7</a> 电脑。不过考量 Windows 7 最早在 2009 年就发表，同时自 2015 年<a href="https://github.com/jtswrk284/ntdtv/blob/master/gb/微软.md">微软</a>就已不再为 Windows 7 开发新功能，用户在转换期之后，或许还是应考虑将电脑换机至 <a href="https://github.com/jtswrk284/ntdtv/blob/master/gb/windows-10.md">Windows 10</a> 机种：</p>
<p>1、确保 Wi-Fi 分享器还能获得更新，同时也应立即更新其至最新版。</p>
<p>2、另外开设一个使用者账号作为日常使用，并关闭其管理者权限。</p>
<p>3、务必继续正常使用防毒软体并定时更新。</p>
<p>4、基本的 Windows 安全功能须保持开启，比如防火墙。</p>
<p>5、关闭或禁用一些不必要且容易有资安疑虑的功能，如 Java、Adobe Flash，一些不常用的程式也最好删掉，尤其是第三方浏览器工具。</p>
<p>6、不要使用 IE 浏览器。</p>
<p>7、备份重要的系统或档案，并独立放在另一个硬碟里，以免电脑在遭受攻击后影响这些档案的使用。</p>
<p>8、尽量更新各类应用程式，比如已确定会继续支援 Windows 7 的 Google Chrome。</p>
<p>（转自《看中国》/责任编辑：文汇）</p>
